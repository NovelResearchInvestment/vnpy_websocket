## 说明

基于websocket-client开发的多线程Websocket API客户端，用于开发高性能的Websocket交易接口。

## 安装

安装环境推荐基于4.0.0版本以上的【[**VeighNa Studio**](https://www.vnpy.com)】。

直接使用pip命令：

```
pip install vnpy_websocket
```

下载解压后在cmd中运行

```
pip install .
```
